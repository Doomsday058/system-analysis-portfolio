# Системный анализ – портфолио кейсов

Демонстрационные кейсы: постановка задачи, проектирование интеграций
и моделей данных, документация решений.

> **Дисклеймер.** Кейсы собраны с нуля на синтетических данных и обобщённых
> сценариях. Не содержат кода, схем, выгрузок или иных материалов
> работодателей. Названия систем условные. Цель – показать подход
> к анализу и проектированию, а не воспроизвести конкретную реализацию.

## Структура каждого кейса

Контекст и проблема → участники и роли → процесс as-is → требования →
целевое решение и архитектура → модель данных → контракты интеграции →
обработка ошибок и граничные случаи → результат → ограничения решения.

## Инструменты

Нотации: BPMN 2.0, UML (sequence, state), ER-моделирование.
Документация: Markdown, Mermaid (diagrams-as-code), OpenAPI 3.0, draw.io.
Схемы для воркшопов со стейкхолдерами – Miro.
